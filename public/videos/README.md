# Videos FME
Coloca aquí los videos del proyecto. Formatos recomendados:
- .mp4 (H.264/H.265) — compatibilidad universal
- .webm (VP9) — mejor compresión en navegadores modernos

Convención de nombres:
- fme-reel.mp4 / fme-reel.webm — reel principal Home
- barrio-loop.mp4 / barrio-loop.webm — loop de fondo Barrio
- manifesto-clip.mp4 — clip sección Manifiesto
